# Middlebury Menu API
An REST API for menus.middlebury.edu.

_Updated 3/2/14_

## Endpoints

API base url: `middmenuapi.herokuapp.com`.

```
/
```
Returns the current day's menu.

```
/yyyy-mm-dd
```
Returns the menu for `yyyy-mm-dd`.  

## Development

Setup:
```sh
$ npm install
```

Run the server:
```sh
$ node app.js
```
